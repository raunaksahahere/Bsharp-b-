B# Programming Language

B# is a beginner-friendly programming language that uses structured English.
It runs using its own interpreter.

How to Run:

1. Install Python 3 (only to run the interpreter itself).
2. Open terminal inside this folder.
3. Run:

   python bsharp.py sample.bsharp

You can create your own .bsharp files and execute them.

Features Currently Included:
- Variables (let, change)
- Printing (say)
- Basic arithmetic (plus)
- Comments (note)
